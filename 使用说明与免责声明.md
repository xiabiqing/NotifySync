# NotifySync 使用说明与免责声明

## 一、软件用途
NotifySync 用于将安卓手机通知同步到 Windows 电脑端，提升消息查看效率。

适用场景示例：
- 实习/工作沟通消息（如 Boss、企业应用等）
- 二手平台消息（如闲鱼等）
- 验证码、系统提醒等需要及时查看的通知

---

## 二、安装与准备

### 1）Windows 端
1. 运行 `NotifySync.exe`
2. 首次运行若出现防火墙提示，请允许局域网访问
3. 记录软件界面显示的 `IP:端口`

### 2）Android 端
1. 安装 `NotifySync.apk`
2. 首次打开后，按提示完成授权：
   - 通知监听权限（必须）
   - 通知发送权限（Android 13+）
   - 建议开启自启动、关闭省电限制（避免后台被杀）

---

## 三、快速连接步骤
1. 确保手机和电脑在同一局域网（同一 Wi-Fi）
2. 在手机端填写电脑端显示的 `IP` 与 `端口`
3. 如已配置 Token/加密密钥，请保持手机端与电脑端一致
4. 发送测试通知，确认连通

---

## 四、核心功能说明

### 1）基础设置
- 服务状态查看
- 复制连接地址
- 应用静默策略（按应用开关“静默/提醒”）

### 2）模式选择（必看）
- **系统通知模式**（默认）：进入 Windows 通知中心
- **隐私模式**：不进入 Windows 通知中心，改为应用内提醒
- 可配置：
  - 弹窗停留时间
  - 弹窗声音
  - 本地消息存储路径
  - 占用检测与一键清理

### 3）安全配置（推荐）
- 安全 Token（鉴权）
- 加密密钥（内容加密）
- 修改后实时自动保存并即时生效

### 4）日志输出
- 查看运行日志
- 快速排查连接、鉴权、通知接收问题

---

## 五、关于“消息留存/垃圾占用”

1. **系统通知模式**下，通知历史由 Windows 通知中心管理。
2. Windows 通知历史可在系统设置中清理（不是无法清理）。
3. **隐私模式**下，本地消息文件由 NotifySync 管理，可检测占用并一键清理。

## 六、常见问题

### Q1：收到了“(无标题)”通知？
部分应用上报通知时可能仅有正文或摘要字段，标题为空属于常见现象。

### Q2：为什么通知偶尔延迟或收不到？
常见原因：
- 手机后台被省电策略限制
- 网络不稳定
- 权限未完整开启

建议：开启自启动、关闭省电限制、保持稳定 Wi-Fi。

### Q3：自启动失败怎么办？
请确认系统权限策略是否限制启动项；必要时使用管理员权限运行后再尝试开启。

## 七、隐私与安全建议
1. 尽量仅在可信局域网中使用
2. 建议始终配置 Token
3. 建议开启加密密钥，避免明文传输
4. 不建议将服务端口暴露到公网
5. 定期更换 Token/密钥
